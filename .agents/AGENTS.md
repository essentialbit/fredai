# FredAI Coding Guidelines

## GitHub Synchronization & Releases
Whenever a change or feature update is completed:
1. **Always update README.md**: Ensure any new features or updates are documented under the changelog, installation, or configuration instructions in [README.md](file:///Volumes/Iron%201TBSSD/Claude/FredAI/README.md).
2. **Synchronize GitHub Releases**: When pushing changes to the repository, ensure a corresponding version tag (following SemVer) or manual release is created on GitHub to keep the release history matching the latest feature updates.

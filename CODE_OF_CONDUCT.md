# Code of Conduct

## Our standard

FredAI is a project about financial intelligence. Discussions should be substantive, direct, and constructive. We welcome people of all backgrounds and experience levels.

Expected behaviour:
- Be direct and specific — vague complaints help no one
- Critique ideas, not people
- Acknowledge when you're wrong or when someone else's approach is better
- Help newcomers get set up — everyone starts somewhere

Not acceptable:
- Personal attacks, insults, or harassment
- Spam or self-promotion unrelated to FredAI
- Sharing or requesting financial advice in the community (Fred is a tool, not an advisor)

## Enforcement

Issues and discussions that violate this standard will be closed or removed. Repeat violations result in a block from the repository.

## Reporting

If you experience or witness unacceptable behaviour, open a private issue labelled `conduct` or contact the maintainer directly via GitHub.
